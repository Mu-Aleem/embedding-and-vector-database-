# Backend Interview Questions (4 Years Experience)

## Node.js Core
1. What is the Node.js event loop and how does it work?
2. What is the difference between `process.nextTick()` and `setImmediate()`?
3. Explain the difference between `require()` and `import`.
4. What are streams in Node.js? Explain readable, writable, duplex, and transform streams.
5. What is the cluster module and how does it help with scaling?
6. How does Node.js handle child processes? Difference between `spawn`, `exec`, `execFile`, and `fork`?
7. What is middleware in Express.js? How does the middleware chain work?
8. What is the difference between `npm` and `yarn` and `pnpm`?
9. How do you handle uncaught exceptions and unhandled promise rejections in Node.js?
10. What are environment variables and how do you manage them across environments?

## APIs & REST
11. What are REST principles? What makes an API RESTful?
12. What is the difference between `PUT`, `PATCH`, and `POST`?
13. How do you version an API?
14. What are HTTP status codes? Explain 200, 201, 204, 301, 400, 401, 403, 404, 500.
15. What is the difference between REST and GraphQL?
16. How do you handle pagination in APIs (offset, cursor-based)?
17. What is rate limiting and how do you implement it?
18. What are idempotent requests and why do they matter?
19. How do you handle file uploads in a REST API?
20. What is HATEOAS in REST?

## Authentication & Security
21. What is the difference between authentication and authorization?
22. How does JWT (JSON Web Token) work? What are access tokens and refresh tokens?
23. What is OAuth 2.0? Explain the authorization code flow.
24. How do you securely store passwords? What is bcrypt and salt?
25. What is CORS? How do you configure it on the server?
26. What are SQL injection, XSS, and CSRF attacks? How do you prevent them?
27. What is HTTPS and how does TLS/SSL work?
28. How do you implement role-based access control (RBAC)?
29. What is session-based auth vs token-based auth?
30. What are API keys and how do they differ from OAuth tokens?

37. What is connection pooling and why is it important?
38. What is an ORM? Pros and cons of using one (Prisma, Sequelize, TypeORM)?
39. What is database sharding and replication?
40. How do you handle database migrations?

44. What is a load balancer and how does it work?

46. How do you design a URL shortener service?
47. What is horizontal vs vertical scaling?
