# Frontend Interview Questions (4 Years Experience)

## HTML
1. What is the difference between `localStorage`, `sessionStorage`, and cookies?
2. What are semantic HTML elements and why are they important?
3. What is the difference between `<script>`, `<script async>`, and `<script defer>`?
4. What are Web Vitals (LCP, FID, CLS) and how do you optimize them?
5. What is the Shadow DOM and how does it differ from the Virtual DOM?

## CSS
6. What is the difference between `em`, `rem`, `px`, `vh`, and `vw`?
7. Explain the CSS Box Model. What is the difference between `content-box` and `border-box`?
8. What is the difference between `position: relative`, `absolute`, `fixed`, and `sticky`?
9. How does Flexbox differ from CSS Grid? When would you use each?
10. What is CSS specificity and how is it calculated?
11. What are CSS custom properties (variables) and how do they differ from SASS variables?
12. Explain the difference between `display: none`, `visibility: hidden`, and `opacity: 0`.
13. What are pseudo-elements and pseudo-classes? Give examples.
14. How do media queries work? What is a mobile-first approach?
15. What is the `z-index` stacking context and how does it work?

## JavaScript Core
16. Explain the difference between `var`, `let`, and `const`.
17. What is hoisting in JavaScript?
18. Explain closures with a practical example.
19. What is the event loop? How do microtasks and macrotasks differ?
20. What is the difference between `==` and `===`?
21. Explain `this` keyword in different contexts (global, object, arrow function, class).
22. What are Promises? Explain `Promise.all`, `Promise.race`, `Promise.allSettled`, and `Promise.any`.
23. What is the difference between `map`, `filter`, `reduce`, and `forEach`?
24. What is event delegation and why is it useful?
25. Explain prototypal inheritance in JavaScript.
26. What is the difference between shallow copy and deep copy? How do you achieve each?
27. What are generators and iterators in JavaScript?
28. Explain `call`, `apply`, and `bind` with examples.
29. What is debouncing and throttling? When would you use each?
30. What is the difference between synchronous and asynchronous programming?

## ES6+ Features
31. What are destructuring, spread, and rest operators?
32. What are template literals and tagged templates?
33. Explain `async/await` and how it differs from `.then()` chains.
34. What are JavaScript modules? Difference between `import/export` and `require/module.exports`?
35. What are `Map`, `Set`, `WeakMap`, and `WeakSet`?

## React (or Framework-Specific)
36. What is the Virtual DOM and how does reconciliation work?
37. Explain the component lifecycle in React (class and functional).
38. What are React Hooks? Explain `useState`, `useEffect`, `useRef`, `useMemo`, and `useCallback`.
39. What is the difference between controlled and uncontrolled components?
40. What is prop drilling and how do you solve it (Context API, Redux, Zustand)?
41. What is React.memo and when should you use it?
42. Explain the difference between `useEffect` and `useLayoutEffect`.
43. What are Higher-Order Components (HOC) vs Render Props vs Custom Hooks?
44. How does React handle keys in lists and why are they important?
45. What is lazy loading and code splitting in React?

## Performance & Optimization
46. How do you optimize a slow React application?
47. What is tree shaking and how does it work?
48. Explain Critical Rendering Path (CRP) in the browser.
49. What are Web Workers and Service Workers?
50. How does browser caching work (ETag, Cache-Control, expires)?

52. What is XSS and CSRF? How do you prevent them?
53. Explain the difference between SSR, CSR, SSG, and ISR.
54. What is a Progressive Web App (PWA)?
